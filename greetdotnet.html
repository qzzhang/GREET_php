A fresh design for GREET life cycle analysis tool

GREET 2015 provides the user with an easy to use and fully graphical toolbox to perform life cycle analysis simulations of alternative transportation fuels and vehicle technologies in a matter of a few clicks. This new tool includes the data of the GREET model, a fast algorithm for processing it and an interactive user interface. The interface allows faster development using graphical representation of each element in the model, and drag & drop editing approach to add and modify data.
