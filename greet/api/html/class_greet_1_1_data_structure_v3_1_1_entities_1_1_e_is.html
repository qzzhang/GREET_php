Detailed Description

Class used to store the EI calculated for each of the mode for each of the fuels for each of the transported resource. It was decided that allowing users to override the EI was too fragile for the code and fuzzy to save to the data file. How to save the overriden EIs, does it worth the development as no-one uses this feature, how to reload the override EIs as the preprocessing of the data create new EI results objects for each mode...

Therefore we fix this object as a unmutable result object that the user can see but not edit
