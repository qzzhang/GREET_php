Dictionary<int, IValue> Greet.DataStructureV3.Interfaces.IResults.OnSiteEmissionsGroups ( IData  data)

This object represents groupped on site emissions results urban only, for example gases that are members of the greenhouse gases group will all be summed up in a single value here.

Emissions that were produced on site for a process or a vehicle, basically it includes anything relevant to the current object except the upstream values

The same groupping is going to be done for other gases groups like criteria pollutants, or vehicle specific emissions

The IDs for each of these groups can be matched with the collection of groups available in IData.GasGroups
