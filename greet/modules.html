Modules

Greet offers three modules by default that are distributed with the main software. These modules are using our API to interact with the core of the GREET software

October 28, 2013
Database Merging Module
Sabbisetti Raja

The database merging module allows to merge different databases together. The principal use of this feature is to allow the user to merge the most recent updates with their database automatically. A secondary use is to allow collaborative work by having different researchers working on different parts of the database and merging their data.

The library is included in the distribution from GREET 2013 release. For more information please see the user guide.

October 28, 2013
Jet Fuels Module
Sokolov Vadim

The jet fuel module offer a model for aircraft and jet fuel production that calculates Well To WAke emissions and energy use. This modules provides the user to add/edit parameters for different types of aircrafts and uses the upstreams of the well to pump pathway of greet to calculate full life cycle results.

The library is included in the distribution from GREET 2013 release. For more information please see the user guide.

October 28, 2013
Stochastic Module
Dieffenthaler David

The stochastic module allows the user to define distributions for parameters in GREET and run stochastic simulations. The tool actually contains 13 distribution types and 3 sampling methods. Frequency analysis tools are also provided and available to explore the distributed parameters and results of the model.

The module itself and underlying data has greatly been improved in the 2014 release providing with a better UI and updated distribution definitions.

The library is included in the distribution from GREET 2013 release. For more information please see the user guide.

September 30, 2014
CCLUB Module
Dane McFarlane

The CCLUB module provides a user interface and a link between the CCLUB (Carbon Calculator for Laud Use Change from Biofuels) and the GREET software. Users are now able to calculate easily the CO2 emissions from land use change and include that into their simulations without the trouble of navigating though all the spreadsheets of the Excel file.

The library is included in the distribution from GREET 2014 release. For more information please see the user guide.

September 30, 2014
Marine Module
Vadim Sokolov

The Marine module allows to calculate WTH (Well to Hull) emissions for different vessel types and trips in the U.S. and international waters. The tool allows to modify ships parameters, segments of the trip and uses the GREET upstream to complete the life cycle of the fuel used by bulk carrier, container vessels and oil tankers.

The library is included in the distribution from GREET 2014 release. For more information please see the user guide.
