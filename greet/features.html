Navigate through the model

Navigating fuel production pathways is achieved using the well to product explorer.


well to product explorer

Through the well to product explorer each individual process in a pathway can be expanded to expose details of inputs and technologies used.

Users can view and choose upstreams for each input, access technologies and emission factors, and examine process fuel shares.

Building complex pathways within minutes

Building a pathway in GREET.net is now accessible to any user without the complexity of linking hundreds of equations manually in the model. Our new graphical pathway editor allows you to simply drag & drop the processes you want in a pathway, save it and explore the results in a matter of a few minutes.

Building a new Pathway

Currently GREET.net includes more than 300 processes to choose from. Mixes of pathways can also be created to weight the individual pathways for a given product.

Add your own data

Users can add/modify resources, technologies, processes, pathways and vehicles using intuitive editors.


Transportation Process Editor

Building a new transportation process can be achieved by drag & drop of transportation modes and locations. These items can then be linked by drawing lines connecting them.

Building a stationary process can be achieved by drag & drop of resources to specify inputs and outputs. Quantities can be entered in virtually any units, and the results are presented in your preferred set of units.


Stationary Process Editor

Choose your prefered units system

The user interface comes with a units system allowing the user to interact with the data using their preferred choice, in addition to performing conversions.

The same inputs can be defined using different units or quantities. Mass, Volume or Energy are automatically converted to perform the simulation.

Analyze results

GREET.net offers multiple levels of results. Users can view process level results with and without upstream included. Users can also see the well to product results by examining the last process in the fuel production pathway with upstream included.


Well to process results

The available results for a process include upstreams from the wells up to and including this process in the pathway. Each individual natural resource used and emissions can be traced in the pathway.


Well to wheels results

The well to wheels results, including the vehicle operation, can be viewed on a per mile basis. The results of different vehicles can be viewed side by side for comparison.

Share your project

All of the model's data is stored in a single file that is easy to share and archive.



Data is stored in a XML structured file

The data file contains the necessary information for simulation, as well as the units used, images and notes for individual entities.

The file stores both default data supplied by the model and values defined by the user. When sharing a data file with others, the modifications can be traced and reverted.
